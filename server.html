📋 快速启动指南

请按照以下步骤启动本地服务器：

方法1：使用VS Code Live Server扩展（推荐）

  1. 在VS Code中安装"Live Server"扩展
  2. 右键点击index.html文件
  3. 选择"Open with Live Server"
  4. 浏览器会自动打开并显示网站

方法2：使用Python简单服务器

  1. 打开命令提示符（cmd）
  2. 导航到项目目录：cd "C:\Users\Ｐanasonic 　CF-SV9\wangzhan"
  3. 运行：python -m http.server 3000
  4. 在浏览器中访问：http://localhost:3000

方法3：使用Node.js服务器

  1. 安装Node.js（如果还没有安装）
  2. 运行：npx http-server -p 3000
  3. 在浏览器中访问：http://localhost:3000
